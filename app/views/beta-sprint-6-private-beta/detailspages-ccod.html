Dataset

Commercial and Corporate Ownership Data

Download data about registered land and property in England and Wales owned by UK companies.

You need to create an account or sign in to download this dataset.


Last updated on the 18 October 2018 (updated monthly).
File size is 25GB


There is no cost for using this dataset but there is a licence.



Read the technical specification for Commercial and Corporate Ownership Data.


How you can use this data

Commercial and Corporate Ownership Data is released under licensed terms of use. These terms also set out the conditions for the re-use of data (in this case, addresses) which contains third party intellectual property rights.

Property addresses contained within the Commercial and Corporate Ownership Data are processed against Ordnance Survey’s AddressBase Premium product, which incorporates Royal Mail’s PAF® database (Address Data).

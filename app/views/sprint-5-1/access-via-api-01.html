About the API

What you can do with the API programmatically

Developers can access the data catalogue with the metadata API, as an alternative to the using the web interface for this service. The API provides both RESTful and functional interfaces, all in JSON format, making it suitable for a wide range of clients.

The base URL for the API is: https://something.gov.uk/api

What you can access

Not all of our datasets are available through our API access. Datasets that have [xxxx] licence attached cannot be accessed using the API.

API access is however available for following datasets:

  • INSPIRE Polygon Data
  • Registered Leases Data
  • Overseas Companies Ownership Data

How licences work with API access

Generating an API key

Once you have created an account and have signed-in, you can navigate to the API key generation screen.

By agreeing to both the licence agreement and the terms of use, you can generate an API key.

Full documentation

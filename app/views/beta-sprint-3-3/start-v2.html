Get land and property data from HM Land Registry

Use this service to download land ownership data for England and Wales.

Looking for information about 1 address? Try this service.

These data files are shared by HM Land Registry.

Most of the data is free and open (available to everyone) but a small number of data files have a licence fee.

Each data file shows information:

  • for every address in England and Wales

  • in tabular (spreadsheet) or shapefile (shapes on a map) format

Download HM Land Registry data

HM Land Registry publishes data files about land and property on this page.

These files show ownership data:

  • for every address in England and Wales (where available)
  • in tabular (spreadsheet) or shapefile (shapes on a map) formats

Looking for information about 1 address? Search by postcode to find out who owns a property and if there’s a mortgage on it.

 

Most of this data is free and open (everyone can use it) but a small number of data files have a stricter licence. This means that you can use the data, but you can only use it in a certain way. If you want to download data with stricter licensing terms, you need to create a free account.
